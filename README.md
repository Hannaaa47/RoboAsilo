# RoboAsilo

## Indice
sdfsd

## Descripción
Robot vigilante para estar de guardia en la robobatalla.

Es un robot para la materia de Mecatronica aplica, el robot va a trabajar en un asilo detectando personas caídas o si tienen alguna emergencia que puedan hacer alguna señal al robot para enviar la alerta a una aplicación mientras el robot patrulla dentro de un área,  va a seguir una línea el robot que es lo que a nosotros se nos hacía más fácil,  tiene que trabajar en automático.

## Componentes:
- Arduino mega
- Cámara es una huskylens
- Seguidor de línea qtr-8a
- 3 sensores ultrasónicos
- Motores con encoder integrado bts7960
- Sensor imu mpu-6050
- Batería de 12v a 7ah
- Power bank

## Funcionalidades
Tenemos un arduino mega que qus queremos se conecte con una aplicación por bluetooth
La cámara es una huskylens
También un seguidor de línea que es un qtr-8a
3 sensores ultrasónicos porque tiene que evadir obstáculos
Los motores son motores con encoder integrado y un sensor imu mpu-6050 para la orientación
Compramos una batería de 12v a 7ah
Y una batería power bank para conectar el arduino
Los drivers de los motores son unos bts7960
